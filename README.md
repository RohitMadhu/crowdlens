# CrowdLens

CrowdLens extracts place crowd and busy signals from the live Google Maps page.

The current implementation uses:

- persistent `thorium-browser`
- `Xvfb`
- raw Chrome DevTools Protocol
- `crowdlens.mjs`

Primary docs:

- `BROWSER_RUNTIME.md`
- `DEPLOYMENT.md`

The supported runtime here is the Thorium + raw CDP path.
